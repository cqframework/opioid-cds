6.0.0 FHIR 3.3.0 Artifacts defined as part of the CDC Opioid Prescribing Guideline Implementation Guide

NOTE: this implementation guide uses STU3 resources. Therefore, the following FHIR 3.3.0 artifacts are only available for download or as raw xml in your browser.

NOTE: as of 3/18/2018, validation frameworks for FHIR 3.3.0 resources don't seem to be current. This has caused issues when trying to POST these artifacts to an FHIR 3.3.0 FHIR server.

The following FHIR 3.3.0 artifacts represent the behavior for implementing CDC Opioid Prescribing Guidelines:
